My Python Notes App

This is a feature-rich note-taking application built using Python and PySide6. It allows users to create, edit, and manage rich-text notes with advanced formatting, image insertion, and checkbox support. Users can assign categories, set statuses, schedule reminders, and organize their notes efficiently using a dynamic sidebar and filtering system.

The app supports dark and light themes, note export/import functionality, and image resizing directly within the editor. Whether you're managing to-do lists, writing tutorials, or documenting aerospace research, this app adapts to your workflow.

My Python Notes App

This is a professional-grade desktop notes application built using Python and PySide6. It enables users to create and manage rich text notes with images, checkboxes, due dates, and reminders. Notes can be categorized and organized for easier filtering.

The app features powerful editing capabilities including text formatting, inline image pasting and resizing, as well as support for light/dark/blue themes. Notes are stored in a local SQLite database with options to import/export your collection.
